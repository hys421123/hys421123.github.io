Java web

发表于 2017-09-02 | 分类于 web

这篇文章罗列的是java web 开发遇到的问题及解决方法。

1. 添加library后，依然报错org/hibernate/cfg/Configuration class not found

解决： 需要把.jar文件包 添加到WebRoot-> WEB-INF ->lib里才能有效识别11

2. JSP中用 javabean传递中文乱码的解决方法

3. tomcat找不到Unable to compile class for JSP

解决： tomcat jdk设置问题

4. The superclass “javax.servlet.http.HttpServlet” was not found on the Java Build Path

解决： 添加Server Runtime库即可

5. The servlets named [x] and [y] are both mapped to the url-pattern[z] which is not permitted”

解决： annotation注解和url-pattern映射属于重复配置，去掉annotation即可。
实际上，可以用注解代替url-pattern 的配置，效果更好。但是，两者只能有其一。

6. Servers重启

没有重启Servers时，很多配置保存再服务器中，导致程序不能完全清空重新启动。

7. Cannot acquire J2EEFlexProjDeployable object for module test [EXPLODED on MyEclipse Tomcat v7.0]

解决： Tomcat部署项目异常，要去掉(remove) server下之前存在但是关闭的项目。
