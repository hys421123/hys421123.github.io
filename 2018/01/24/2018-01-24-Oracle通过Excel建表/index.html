Oracle通过Excel建表操作说明

发表于 2018-01-24 | 更新于 2018-01-24 | 分类于 数据库

Oracle数据库导入操作

  • 图1 Excel中表数据
  • 表1 Excel建表字段




    字段名 字段类型 是否为空 字段注释
    OAE001 hNUMBER(22) N 唯一索引
    AAC051 NUMBER(22) N 人员变更流水号
    AAC001 VARCHAR2(14) N 个人编号
    AAC061 NUMBER(22) Y 申报流水号
    AAE035 DATE(7) N 变更日期

操作步骤

  1. 将上述建表数据复制粘贴到Excel文档中；
  2. 编辑 Excel表格，让Excel数据格式与Oracle中Columns的数据格式保持一致，下图通过插入空列使两者格式一致；
  • 图1 修改的Excel表
  • 图2 对应数据库数据表格式
  1. 修改Excel数据，将不符合规范的数据类型改正，如DATE(7) -> DATE。
  2. 打开PL/SQ Developer的Oracle数据库工具，File-> New-> Table新建数据表，将Excel中的数据复制粘贴到数据库Columns栏目中。
  3. 点击View SQL，添加其他设置语句。

    alter table T_tableName
    add constraint PK_T_tableName primary key (id)
    using index
    tablespace CN185
    pctfree 10
    initrans 2
    maxtrans 255
    storage
    (
    initial 64K
    next 1M
    minextents 1
    maxextents unlimited
    );

  1. 点击Apply，成功建表。(注释：若有报错，改正错误后再应用即可)
