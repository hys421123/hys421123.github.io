Hexo基础命令

hexo init 重建博客网站初始化
hexo server -p 5000 更改端口号为5000
hexo g (generate简写) 生成部署网站
hexo s (server) 启动服务器
hexo clean 清除缓存文件(db.json)和已生成的静态文件(public)
hexo p (publish) 发布
hexo d (deploy) 部署网站
